Add Media

Media files (i.e., audio, video) can go in the media folder in the root directory. Inside that directory are directories for audiobooks, music, and videos: 


--| media
----| audiobooks
----| music
----| videos

Media.js

The media.js file (below) contains the JSON file that is used to propagate the Media Player. Each media item requires 4 pieces of information: 


"type": ["audio", "video", "book"] // Used for filtering on the main page
"tag": ["audio", "video"] // the HTML tag to be created
"title": [string] // Any user-generated string
"url": [string] // The path to the media file
"extension": ["mp3", "mp4"] // The file extension for the media
